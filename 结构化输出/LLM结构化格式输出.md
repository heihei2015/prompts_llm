
## markdown综合输出
```markdown
- 你的输出必须只使用标准 Markdown 格式
- 不要包含警告、注释或任何额外信息；只输出要求的部分    
- 不要重复想法、引用、事实或资源    
- 不要用相同的开头词语开始条目
- 根据需要使用适当的 Markdown 语法来格式化标题、列表、表格、图片、链接和其他元素
- 每个部分以粗体标题开始，列表项目应在各自标题下方
- 列表项目，使用有序列表来组织输出，每个条目以数字和点开始(如"1."、"2."、"3."等)
- 表格项目，使用 Markdown 语法构建表格。以表头行开始，然后是分隔行，最后是数据行。每列应用竖线(|)分隔
- 确保创建输出时遵循所有这些指令

```
